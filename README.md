# Ikoni-texturepack
Православный текстур пак для маинкрафт.
Для корректного отображения всех визуальных эффектов (таких как соединение текстур и кастомное освещение) рекомендуется установить **OptiFine**.
> **Важно:** Скачивайте OptiFine только с официального сайта: [https://optifine.net/downloads](https://optifine.net/downloads)
## Как понять какую версию стоит скачивать?
Первая цифра (Major: 1.0.0 -> 2.0.0): Глобальные изменения. Например, мы полностью перерисовали пак с нуля в другом стиле.

Вторая цифра (Minor: 1.0.0 -> 1.1.0): Добавление нового контента. Например, мы добавили 20 новых икон или изменили GUI.

Третья цифра (Patch: 1.0.0 -> 1.0.1): Минорные исправления. Например, поправил одну кривую текстуру или изменил pack.mcmeta.

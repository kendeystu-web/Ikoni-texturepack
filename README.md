# Ikoni-texturepack
Православный текстур пак для маинкрафт.
На версии 1.21.1 и более могут быть проблемы с текстурами.
Для корректного отображения всех визуальных эффектов (таких как соединение текстур и кастомное освещение) рекомендуется установить **OptiFine**.
> **Важно:** Скачивайте OptiFine только с официального сайта: [https://optifine.net/downloads](https://optifine.net/downloads)
## Как понять какую версию стоит скачивать?

Первая цифра (Major: 1.0.0 -> 2.0.0): Глобальные нововведения и развитие текущего стиля

Вторая цифра (Minor: 1.0.0 -> 1.1.0): Добавление нового контента.

Третья цифра (Patch: 1.0.0 -> 1.0.1): Минорные исправления.
